# openclaw-web-search

Free, privacy-focused web search skill for OpenClaw using DuckDuckGo HTML parsing and GitHub API.

## Features

- 🦆 **DuckDuckGo Search** - No API key required
- 🐙 **GitHub API** - Search issues and code
- 🔒 **Privacy First** - No tracking
- 🆓 **Completely Free** - No usage limits

## Installation

Copy the `web-search` skill folder to your OpenClaw skills directory.

## Usage

```bash
python3 scripts/search.py --query "your search terms"
```
